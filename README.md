Project Title Data Visualization – Histogram & Bar Chart

Description This project demonstrates how to visualize data distributions using:

A Histogram for continuous variables
A Bar Chart for categorical variables

It demonstrates:

Data loading using Pandas
Data visualization using Matplotlib
Creating a histogram for continuous data
Creating a bar chart for categorical data
The dataset is processed and visualized using Python libraries in Google Colab.

Tools and Technologies Used:

Python
Pandas
Matplotlib
Google Colab

Visualizations Implemented:

Histogram for continuous data
Bar chart for categorical data
